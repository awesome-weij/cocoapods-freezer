# cocoapods-freezer

cocoapods-freezer is a plugin of cocoapods. It uses for cache of intergation!

## Installation

    $ gem install cocoapods-freezer

## Usage

Configurate such as example, then call 'pod install'.
	
``` ruby
#use_freezer!

target 'Demo' do
  pod 'AFNetworking'
end

```
